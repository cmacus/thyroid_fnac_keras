# thyroid_fnac_keras

A keras model to classify thyroid FNAC microphotographs (512 x 512) into 'benign' or 'malignant'.

Included
* data512 folder containing train and test images in both 10x and 40x magnification
* two set of weights at 10x magnification (first_try and second_try)
* one set of weights at 40x magnification
* model.json
* demo images

Here is the publication : http://www.jpathinformatics.org/article.asp?issn=2153-3539;year=2018;volume=9;issue=1;spage=43;epage=43;aulast=Sanyal;type=0; try improving the score
